# PIC16F887 Level Control System

Embedded level monitoring and control system developed using the PIC16F887 microcontroller.

The system reads an analog level signal through the ADC module and compares it against a user-defined setpoint entered from a matrix keypad.

## Features
- Analog level acquisition using ADC
- User-defined setpoint
- Matrix keypad interface
- LCD monitoring interface
- Real-time comparison logic
- Status indication

## Hardware Used
- PIC16F887
- 4x4 Matrix Keypad
- 16x2 LCD
- Potentiometer / analog input
- CCS C Compiler

## Software
- Embedded C
- CCS C Compiler

## Main Concepts
- ADC
- Embedded control systems
- Human-machine interface
- Real-time monitoring
- State logic

- `level_control_system.c` → Main application
- `LCD1.c` → LCD driver library
